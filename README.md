## 📌 About the Project

**JellyJam** is an open-source web application inspired by Spotify Jam, designed to work seamlessly with **Jellyfin** media servers. It allows users to create synchronized, real-time listening sessions, enabling friends to stream, queue, and enjoy their self-hosted music library together perfectly in sync. 

Built with a modern full-stack architecture, it leverages Jellyfin's powerful API to manage media playback while providing an interactive and collaborative social experience.

> ⚠️ **Note:** This project is currently under active development (**Work in Progress**). Features are being implemented, and the codebase is evolving rapidly.

---

## 🛠️ Tech Stack

The project is split into a decoupled full-stack architecture:

* **Backend:** Built with **NestJS** (Node.js framework) to handle real-time synchronization, session management, and seamless integration with the **Jellyfin API**.
* **Frontend:** Built with **React** to deliver a fast, responsive, and intuitive user interface for managing queues and playback.
